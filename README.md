# ⟐Akashic.one Server  
**⟐dot.one Network Core Server**  
Self-hosted backend powering all applications within the AetherPulse.one ecosystem. Handles data, APIs, and secure app communication across every ⟐dot.one project.  
---

### Features  
- Secure HTTPS connection via Caddy  
- Automatic dynamic DNS updates using Now-DNS  
- Local SQLite database for app data storage  
- Auto-start and background operation through Task Scheduler  
- Passkey and nonce authentication system  
- Offline capable and fully independent from third-party hosts  
- Central hub for ⟐Sync.one, ⟐Orbit.one, and all future ⟐dot.one apps  
- Designed for 24/7 uptime and self-recovery  

---

## Tech Stack  
- Node.js (Express)  
- SQLite (Local Database)  
- Caddy (HTTPS)  
- Now-DNS (Dynamic DNS)  
- Windows Task Scheduler  
- JSON-based Data Handling
